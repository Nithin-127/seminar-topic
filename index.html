Semantic UI Button Example

This section showcases various button styles from Semantic UI. Semantic UI is a modern CSS framework using natural language principles and human-readable class names for intuitive HTML and pre-styled components.

Note on External Frameworks:

CSS Grid is a native browser feature for layout. Semantic UI, however, is a framework that requires its own CSS files to be linked in the HTML document to apply its predefined styles to elements like buttons. The CDN link included in the `<head>` section facilitates this, as the provided sources only show examples of using Semantic UI classes, not the full framework's CSS.
